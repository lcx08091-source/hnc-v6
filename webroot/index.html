TC 命令参考
# 下行限速 (egress on wlan2)
tc qdisc add dev wlan2 root handle 1: htb default 999
tc class add dev wlan2 parent 1: classid 1:14 \
  htb rate 10mbit ceil 10mbit

# 上行限速 (ingress via IFB)
tc qdisc add dev wlan2 handle ffff: ingress
tc filter add dev wlan2 parent ffff: protocol ip \
  u32 match u32 0 0 action mirred egress redirect dev ifb0

# 延迟注入 (netem)
tc qdisc add dev wlan2 parent 1:14 handle 101: \
  netem delay 200ms 20ms loss 1%

关于 DPI 模块

HNC DPI (hnc_dpid) 是被动的元数据观察 daemon, 用 AF_PACKET + cBPF 抓 DNS 和 TLS ClientHello 元数据，并用内置/导入规则做只读 L3 标签/分类，不做任何主动操作 (不 NFQUEUE、不 DNS 劫持、不改 iptables/tc).

当前 rc19/L2 阶段按客户端聚合 DNS 与 TLS SNI 元数据，只做域名画像，不做 App 分类、不参与限速。

如果卡在“等待接口就绪”，可点上方 重新绑定 DPI；规则库可在本页导入到 /data/local/hnc/etc/dpi_rules.json；关闭抓包: 修改 /data/local/hnc/etc/dpi_config.json 设 "disable_capture": true 后重启 watchdog.
